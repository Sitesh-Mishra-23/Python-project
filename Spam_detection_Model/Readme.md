# 📧 Spam Detector

A machine learning app that detects whether a message is Spam or Not Spam.

## 🛠️ Tech Stack
- Python
- Scikit-learn (Naive Bayes + TF-IDF)
- Streamlit

## 📊 Accuracy
~97-98% on SMS Spam Collection dataset

## 🚀 How to Run
pip install -r requirements.txt
python model.py
streamlit run app.py

## 📁 Dataset
SMS Spam Collection Dataset from Kaggle
